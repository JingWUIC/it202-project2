Instructions

Click on drop-down list and select to search by ward or status. If you choose to search by ward make sure you type in a positive integer.
Otherwise, if you choose to go by status, type in a string either being Completed or Open.
Then, click on the navigation tabs at the top, if you want to view it by List View then click on List otherwise, click on Map for the Map View.

About Page

The name of this app is called Graffiti Locator. It can located graffiti in the Chicago area and is viewable on the map or as a list.
It also gives the location and status of the graffiti whether it has been cleaned up or not.
